## How to Upgrade

```bash
# Auto-upgrade to the latest version
opencode upgrade

# Check current version
opencode --version
```

OpenCode has auto-update enabled by default and usually requires no manual action. To disable auto-update, set in your config file:

```json
{
  "autoupdate": false
}
```

## Version Notes

- **Major version** (e.g., 1.x.x → 2.x.x): Major updates, may contain breaking changes
- **Minor version** (e.g., 1.1.x → 1.2.x): New feature releases
- **Patch version** (e.g., 1.1.10 → 1.1.11): Bug fixes and minor improvements
