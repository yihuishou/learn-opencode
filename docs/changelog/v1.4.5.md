# v1.4.5：OTLP Span 导出 + Question API 开放

**发布日期**：2026年4月15日

---

## 🎯 一句话总结

AI SDK 遥测 Span 可导出到 OTLP 后端，Question API 的 Schema 和 OpenAPI 规范公开暴露。

---

## 🆕 新功能：OTLP Span 导出

### 这次更新了什么？

AI SDK 的遥测 Span（traces）现在可以导出到 OTLP 后端。

### 这有什么用？

之前只能导出基本的遥测数据，现在可以将每次 AI 调用的详细 Span（请求耗时、Token 消耗、模型信息等）发送到 Grafana Tempo、Jaeger 等 OpenTelemetry 后端进行可视化分析。

---

## 🆕 新功能：Question API 开放

### 这次更新了什么？

暴露了实验性的 Question API 的 Schema 和 OpenAPI 规范，以及可复用的 Question Handler 工厂。

### 这有什么用？

开发者可以基于这些规范构建自己的 Question API 服务，或集成到自定义工具中。

---

## 🔧 问题修复

### Desktop Shell 命令目录

**问题**：Desktop 版的 Shell 命令没有从用户主目录启动。

**修复**：现在默认从主目录启动。

### Desktop 初始化弹窗

**问题**：全局同步初始化时会弹出不必要的错误提示。

**修复**：全局同步初始化期间不再弹出错误弹窗。
