## 如何升级

```bash
# 自动升级到最新版本
opencode upgrade

# 查看当前版本
opencode --version
```

OpenCode 默认开启自动更新，通常无需手动操作。如需关闭自动更新，可在配置文件中设置：

```json
{
  "autoupdate": false
}
```

## 版本说明

- **主版本号**（如 1.x.x → 2.x.x）：重大更新，可能包含不兼容变更
- **次版本号**（如 1.1.x → 1.2.x）：新功能发布
- **修订号**（如 1.1.10 → 1.1.11）：Bug 修复和小改进
