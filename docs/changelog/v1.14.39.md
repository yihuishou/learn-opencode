# v1.14.39：Desktop 代理支持 + 存储健壮性

**发布日期**：2026年5月5日

---

## 🎯 一句话总结

Desktop 版现在支持 HTTP 代理环境变量，存储读取失败时不再报错而是安全返回。

---

## 🔧 问题修复

### HTTP 代理支持

**问题**：Desktop 版不读取 `HTTP_PROXY`、`HTTPS_PROXY` 等代理环境变量，导致在需要代理的网络环境中无法连接。

**修复**：Desktop 版现在正确使用 `HTTP_PROXY` 及相关代理环境变量。

**对你有什么影响**：如果你的网络环境需要通过代理访问外网（常见于企业内网），Desktop 版现在能正常工作。

---

### 存储读取健壮性

**问题**：Desktop 版无法读取某个存储值时会直接报错。

**修复**：读取失败时返回 `null` 而不是抛出异常。

**对你有什么影响**：存储文件损坏或丢失某些数据时，应用不会崩溃，而是优雅降级。

---

## 📊 要不要升级？
