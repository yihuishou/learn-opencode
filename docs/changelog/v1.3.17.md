# v1.3.17：Cloudflare 配置引导 + Windows 终端输入修复

**发布日期**：2026年4月6日

---

## 🎯 一句话总结

Cloudflare Workers AI 设置现在会引导填写必要信息，Windows 终端的 kitty 键盘处理恢复默认。

---

## 🆕 改进：Cloudflare 配置引导

### 这次更新了什么？

使用 Cloudflare Workers AI 或 AI Gateway 时，如果缺少必要的环境变量，现在会：
- 提示你输入缺失的账户信息
- 显示清晰的配置错误说明

### 这有什么用？

之前配置不全会直接报模糊的错误。现在会明确告诉你需要设置哪些变量，一步步引导完成配置。

## 🔧 问题修复

### Windows 终端输入

**问题**：之前为解决兼容问题做的 kitty 键盘 workaround 在回退后，Windows 终端上出现了输入问题。

**修复**：恢复 Windows 终端的默认 kitty 键盘处理。
