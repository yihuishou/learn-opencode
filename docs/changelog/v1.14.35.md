# v1.14.35：Diff 渲染边界修复

**发布日期**：2026年5月5日

---

## 🎯 一句话总结

修复了当文件内容包含 `diff --git` 文本时，会话 Diff 渲染可能出错的问题。

---

## 🔧 问题修复

### Diff 补丁边界保留

**问题**：当会话中的文件内容本身包含 `diff --git` 文本时（比如在讨论 Git 补丁或查看 `.patch` 文件），Diff 补丁的边界信息会被错误解析，导致会话 Diff 无法正常渲染。

**修复**：保留 Diff 补丁的边界标记，确保渲染时正确识别。

**对你有什么影响**：如果你在对话中让 AI 处理过包含 Git 补丁格式的文件，之前的 Diff 显示可能有问题，升级后恢复正常。
