# v1.4.6：快照长文件列表修复 + Desktop 提交状态修复

**发布日期**：2026年4月15日

---

## 🎯 一句话总结

修复了快照阶段处理超长文件列表时的性能问题和 OTEL Header 解析，Desktop 版修复了提示词提交和会话标题编辑的状态问题。

---

## 🔧 问题修复

### 快照超长文件列表

**问题**：当项目包含大量文件时，快照的 staging 阶段性能很差。

**修复**：优化了长文件列表的 staging 性能。

**对你有什么影响**：大型项目的快照生成更快。

---

### OTEL Header 解析

**问题**：当 Header 值中包含 `=` 字符时，OTEL Header 解析会出错。

**修复**：正确处理包含 `=` 的 Header 值。

---

### Desktop 提交状态

**问题**：Desktop 版的提示词提交时，状态更新不一致导致发送失败。

**修复**：修复了状态更新逻辑。

---

### Desktop 会话标题间距

**问题**：编辑会话标题时，输入框的间距不正确。

**修复**：调整了标题输入框的间距。

---
