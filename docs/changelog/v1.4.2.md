# v1.4.2：子 Agent 点击修复 + Desktop 加载优化

**发布日期**：2026年4月9日

---

## 🎯 一句话总结

修复了子 Agent 在运行时不可点击的问题，移除了 Desktop 版的强制加载延迟。

---

## 🔧 问题修复

### 子 Agent 运行中不可点击

**问题**：v1.3.10 中子 Agent 调用在等待时应该是可点击的，但在 v1.4.x 中又变得不可点击了，直到子 Agent 完成才能点。

**修复**：子 Agent 在运行中现在可以正常点击查看。

---

### Desktop 强制加载延迟

**问题**：Desktop 版在连接服务器时有一个强制的加载等待时间，即使已经连上了也要等。

**修复**：移除了强制延迟，连接完成后立即进入就绪状态。

**对你有什么影响**：Desktop 版启动更快。

---
