# v1.3.9：插件入口修复

## 🎯 一句话总结

修复了插件入口点路径解析问题，改进了 Anthropic 模型选择和文件读取工具描述。

---

## 🔧 问题修复

### 插件入口解析

**问题**：插件入口点（entrypoint）的路径解析对没有前导 `.` 的路径处理不正确，导致某些插件无法正确加载。

**修复**：统一了路径解析逻辑。

**对你有什么影响**：之前加载失败的插件现在可能恢复正常。
