# v1.3.6：变体搜索修复 + Token 统计修正

**发布日期**：2026年3月29日

---

## 🎯 一句话总结

修复了模型变体对话框的搜索过滤功能，修正了 Anthropic 和 Bedrock 的 Token 统计虚高问题。

---

## 🔧 问题修复

### 变体对话框搜索失效

**问题**：v1.3.4 新增的模型变体选择对话框中，输入搜索文字后无法正确过滤列表。

**修复**：搜索输入现在能正确过滤变体列表。

**对你有什么影响**：在变体对话框中输入关键词，列表会实时过滤匹配的选项。

---

### Token 使用量虚高

**问题**：Anthropic 和 Amazon Bedrock 的 Token 使用统计存在重复计算，导致显示的 Token 数量比实际消耗多。

**修复**：消除了重复计算的逻辑。

**对你有什么影响**：会话统计面板中的 Token 数量现在准确反映真实消耗。
